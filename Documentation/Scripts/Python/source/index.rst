Outils Python de pré et post-traitement
=======================================

Bienvenue dans la documentation des scripts de pré et post traitement pour OpenFOAM. Les outils présentés ici servent pour la génération de houle, visualisation et mesure de hauteur d'eau avec placement de sondes, reconstruction de spectre et mesure de débit de franchissement.
