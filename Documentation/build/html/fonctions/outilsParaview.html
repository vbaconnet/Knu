outilsParaview module¶

Description¶

Liste de fonctions permettant d’interagir dans l’environnement paraview.

Fonctions¶

outilsParaview.contour(input, field='alpha.water', value=0.5)¶

Découpe le domaine en une ligne selon l’axe z, puis renvoie le point où alpha.water = 0.5

Paramètres

  • inputElement (dataset (peut être FoamFile ou Slice, Contour, etc...)) – Element parent à partir duquel on va slice

  • field (str, optional) – Variable à partir de laquelle faire le contour, par défaut « alpha.water »

  • value (float, optional) – valeur de la variable à tracer, par défaut 0.5

Renvoie

contour : le contour où field = value

Type renvoyé

paraview.simple.Contour

outilsParaview.xSlice(input, xSliceLocation)¶

Découpe le domaine en une surface dans le plan (y,z)

Paramètres

  • inputElement (dataset (peut être FoamFile ou Slice, Contour, etc...)) – Element parent à partir duquel on va slice

  • xSliceLocation (float) – coordonnée x pour la découpe

Renvoie

xSlice : surface dans le plan (y,z)

Type renvoyé

paraview.simple.Slice

outilsParaview.ySlice(input, ySliceLocation)¶

Paramètres

  • ySliceLocation (float) – coordonnée y pour la découpe

Renvoie

ySlice : surface dans le plan (x,z)

Type renvoyé

paraview.simple.Slice
