Tracé rapide de valeurs¶

Informations¶

  • Fichier : Knu/Scripts/Post-traitement/traceSondes.py

  • Auteur : Victor Baconnet

  • Dernière modification le : 30/08/2021

Description¶

Tracer des données à partir d’un fichier .csv, avec en première colonne le signal temporel. Trace toutes les valeurs sur le même graphique.

Note

Pour un affichage de graphiques détaillé, consultez les fonctions disponibles dans outilsExcel pour exporter vos valeurs vers Excel et générer des graphiques automatiquement

Utilisation¶

$ traceSondes.py <fichier>

<fichier> doit être au format .csv, et doit être formaté comme le modèle suivant:

temps,nom_1,nom_2,...,nom_N
t_1,val_11,val_12,...,val_1N
...
t_N,val_M1,val_M2,...,val_MN

En résumé, les données sont organisées en colonnes, et la première doit être un vecteur de temps.
